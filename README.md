## Présentation
Cette application web propose une galerie de cakes sous forme de cartes interactives.  
Chaque carte affiche :
- Une image
- Un titre
- Un slogan poétique
- Une rangée de cinq étoiles dorées
- Le prix en roupies indiennes (₹)
- Un bouton panier-caddie élégant

## Fonctionnalités
- **Pagination dynamique** : affichage par tranche de 8 cakes, avec navigation par numéros et flèches.
- **Effets visuels** : ombre douce au survol des cartes, mise en valeur de la page active.
- **Design responsive** : grille compacte en CSS Grid.
- **JavaScript dynamique** : insertion des données (images, titres, slogans, prix) directement depuis un tableau JS.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
